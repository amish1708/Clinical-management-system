🏥 Clinic Management System
📋 Project Overview

The Clinic Management System (CMS) is a simple yet functional web-based application that automates daily clinic operations.
It helps manage patient registration, billing, and prescriptions through two key modules — Receptionist Dashboard and Doctor Dashboard.

The system improves efficiency, reduces paperwork, and ensures faster, error-free handling of patient data.

🚀 Features

👩‍💼 Signup & Login System (Authentication)

🧾 Receptionist Dashboard

Register new patients with name, age, and contact info

Generate unique patient tokens

Create bills for patients

View patient and bill lists in real-time

🩺 Doctor Dashboard

View patient list

Add and save prescriptions

Store and manage digital prescription data

💻 Responsive Frontend using HTML, CSS, and JavaScript

🔒 Simple, secure, and scalable structure

🧠 System Modules
Module	Description
Signup Module	Allows users to create new accounts
Login Module	Authenticates users and redirects to dashboard
Receptionist Module	Handles patient registration, tokens, and billing
Doctor Module	Manages prescriptions for registered patients

⚙️ How to Run the Project

Download or clone this repository:

git clone https://github.com/yourusername/clinic-management-system.git


Open the project folder in VS Code or any editor.

Run locally:

Simply open login.html in your browser.

Signup a new user using signup.html.

After login, access the receptionist.html or doctor.html pages.

(Optional) For backend integration:

Connect auth.js and receptionist.js to a Firebase or Node.js server to store user and patient data permanently.
